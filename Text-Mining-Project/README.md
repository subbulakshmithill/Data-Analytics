# Text-Mining-Project
Aspect-Based Sentiment Analysis on a dataset with Hotel reviews
- Data Extraction from website
- Normalization (pre-processing)
- Aspect detection
- TextBlob for assigning polarity
- Training the model with Naive Bayes Classifier.
